# 📋 Lista de Tarefas em Python

Aplicação de linha de comando desenvolvida em Python para gerenciamento de tarefas, permitindo adicionar, listar, concluir e remover tarefas com armazenamento em arquivo local.

---

## 🚀 Funcionalidades

* ✔ Adicionar tarefas
* ✔ Listar tarefas
* ✔ Marcar tarefas como concluídas
* ✔ Remover tarefas
* ✔ Persistência de dados em arquivo `.txt`

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Manipulação de arquivos (`open`, `read`, `write`)
* Estruturas de controle (loops, condicionais)

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/lista-tarefas-python.git
```

2. Acesse a pasta do projeto:

```bash
cd lista-tarefas-python
```

3. Execute o programa:

```bash
python main.py
```
